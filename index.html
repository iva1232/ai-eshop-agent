Store Connection

Enter your Shopify store credentials. Your Admin API token is used only in your browser and is never stored on any server.

Your Shopify store domain
Found in Shopify Admin → Settings → Apps → Develop Apps → API credentials
●Not connected — enter your credentials above

📦 How to Get Your API Token

You need a private app token with product write permissions.

1
Go to your Shopify Admin → Settings
2
Click Apps and sales channels → Develop apps
3
Create an app, then go to API credentials
4
Enable write_products scope under Admin API
5
Install the app and copy your Admin API access token

🔒 Security Notes

Your Shopify token is handled with the same security as your API key.

• Token is stored only in browser memory — never written to disk or sent anywhere except Shopify's own API.
• All API calls go directly from your browser to Shopify.
• Use a private app token with only the permissions you need (write_products).
• Revoke the token at any time in your Shopify Admin.
⚡ n8n Automation
Configure webhook endpoints so DropAI automatically sends AI results to your n8n workflows for full automation.

Webhook Configuration

Set a webhook URL for each tool. When you click "Send to n8n" or enable auto-send in settings, results are posted to these URLs as JSON.

📖 How to Set Up in n8n

Create a Webhook trigger node in n8n to receive DropAI data.

1
Open n8n and create a new workflow
2
Add a Webhook node as the trigger
3
Set method to POST, authentication to None
4
Copy the Webhook URL from n8n into the field above
5
Activate the workflow, then test it from DropAI

📦 Payload Structure

This is the JSON DropAI sends to your n8n webhook on every request.

{
  "source": "DropAI",
  "tool": "listing_generator",
  "timestamp": "2025-01-15T10:30:00Z",
  "input": {
    "product": "Magnetic Car Phone Mount",
    "details": "Black and silver..."
  },
  "output": "TITLE: Magnetic Car Phone...",
  "session_id": "drop_abc123"
